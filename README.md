# Produtos API 

API REST desenvolvida para gerenciamento de produtos, criada como projeto de estudo para dominar o ecossistema Spring Boot.

## Tecnologias Utilizadas

- **Java 17** (ou a versão que estiver usando)
- **Spring Boot 3**
- **Spring Web** (REST API)
- **H2 Database** (Banco de dados em memória)
- **Maven** (Gerenciador de dependências)

## Funcionalidades

- [x] Criação de estrutura inicial do projeto
- [x] Configuração de conexão com Banco H2
- [x] Endpoint de cadastro de produtos (POST)
- [ ] Persistência de dados com JPA (Em andamento)
- [ ] Validações e Tratamento de Erros (Futuro)
